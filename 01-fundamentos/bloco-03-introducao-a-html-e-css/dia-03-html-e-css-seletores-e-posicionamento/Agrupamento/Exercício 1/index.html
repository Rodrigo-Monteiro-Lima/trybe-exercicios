Lista dos Melhores Sites que já vi

Top 3 dos melhores sites que existem na web:

  1. Youtube.com
  2. Spotify.com
  3. Discord.com

Principais motivos para eu gostar deles:

  • Youtube: Podemos assistir vídeos sobre qualquer coisa, não só pra entrenimento, mas para estudo também.
  • Spotify: Não precisamos mais passar pelo trabalho de ter que baixar música por música, pode-se escutar qualquer música.
  • Discord: Uma das principais plataformas de comunicação, principlamente pra quem joga. Chamada de vídeo ou voz com centenas de pessoas de qualquer lugar do mundo.
